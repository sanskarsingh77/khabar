Khabar is a modern news aggregation web application built using React.  
It fetches real-time news using the GNews API and provides category-wise browsing.

## Features
- Category-based news (Business, Technology, Sports, Health, etc.)
- Search news articles
- Load more news (pagination)
- Responsive UI using Bootstrap
- Dark mode support
- Secure API key handling using environment variables
- Deployed on Vercel

## Tech Stack
- React.js
- JavaScript
- Bootstrap
- REST API (GNews)
- Vercel
